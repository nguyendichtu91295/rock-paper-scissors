# Rock Paper Scissors

This is a very old school game made just from HTML/CSS and Javascript.

## Installation

You can use *npm* or *yarn* to install Rock Paper Scissors.

```bash
yarn install
```

## Usage

```bash
yarn start
```

The Game will be running at [http://localhost:3000](http://localhost:3000)
